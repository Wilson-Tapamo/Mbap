# Design System KmerFlow (mobile-first)

## Principes
- 3 actions maximum par écran.
- Boutons larges (min-height 48px).
- Texte non technique, orienté décision.
- Icônes explicites + couleur d'état (succès, alerte, danger).

## Tokens UI
- Radius: 14/18/24.
- Ombres: douces, profondeur légère.
- Typo: Inter (fallback system).
- Couleurs:
  - Primary: #0EA5E9
  - Success: #16A34A
  - Warning: #F59E0B
  - Danger: #EF4444
  - Light bg: #F5F7FB
  - Dark bg: #020617

## Composants clés
- KPI Card (argent/profit).
- Insight Card (coach IA + action CTA).
- Quick Action Button (vente, dépense, produit).
- Stock Alert Row.
- Skeleton Blocks.
- Empty State intelligent (conseil + action simple).

## Micro-interactions
- Animation d'entrée 250–350ms.
- Feedback tactile (pressed state).
- Confirmations courtes: "Enregistré ✅".
